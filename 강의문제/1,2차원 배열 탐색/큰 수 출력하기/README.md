# 큰 수 출력하기

### 문제 설명

<p>N(1<=N<=100)개의 정수를 입력받아, 자신의 바로 앞 수보다 큰 수만 출력하는 프로그램을 작성하세요.(첫 번째 수는 무조건 출력한다)</p>

<h5>입력 설명</h5>

<ul>
<li>첫 줄에 자연수 N이 주어지고, 그 다음 줄에 N개의 정수가 입력된다.</li>
</ul>

<h5>입출력 예</h5>
<table class="table">
        <thead><tr>
<th>n</th>
<th>arr</th>
</tr>
</thead>
        <tbody><tr>
<td>6</td>
<td>[7 3 9 5 6 12]</td>
</tr>
</tbody>
</table>
<h5>출력</h5>
<ul>
<li>[7,9,6,12]</li>
</ul>
