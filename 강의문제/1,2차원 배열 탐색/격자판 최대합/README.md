# 격자판 최대합

### 문제 설명

<p>N*N의 격자판이 주어지면 각 행의 합, 각 열의 합, 두 대각선의 합 중 가 장 큰 합을 출력합니다.</p>

<h5>입력 설명</h5>

<ul>
<li>첫 줄에 자연수 N이 주어진다.(1<=N<=50) 두 번째 줄부터 N줄에 걸쳐 각 줄에 N개의 자연수가 주어진다.</li>
<li>각 자연수는 100을 넘지 않는다.</li>
</ul>

<h5>출력 설명</h5>

<ul>
<li>최대합을 출력합니다.</li>
</ul>

<h5>입출력 예</h5>
<table class="table">
        <thead><tr>
<th>n</th>
<th>arr</th>
</tr>
</thead>
        <tbody><tr>
<td>5</td>
<td>
[
        [10,13,10,12,15],
        [12,39,30,23,11],
        [11,25,50,53,15],
        [19,27,29,37,27],
        [19,13,30,13,19]
]</td>
</tr>
</tbody>
</table>
<h5>출력</h5>
<ul>
<li>155</li>
</ul>
